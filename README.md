# 3D 數位畫廊

這是一個使用 p5.js 製作的 3D 數位畫廊項目。

## 功能特點

- 深黑色背景與透視灰色地面，營造 3D 空間感
- 水平排列的作品縮圖，具有金屬質感邊框
- 點擊縮圖彈出大視窗，使用 iframe 載入作品內容
- 滑鼠左右移動時的視差效果，模擬在畫廊中走動

## 如何運行

1. 確保你有 Python 安裝在系統中
2. 在項目目錄運行：`python -m http.server 8000`
3. 在瀏覽器中打開 `http://localhost:8000`

## 自定義作品

編輯 `sketch.js` 中的 `artworks` 陣列來添加你的作品：

```javascript
let artworks = [
  { title: "你的作品標題", thumbnail: "縮圖URL", url: "作品完整URL" },
  // 添加更多作品...
];
```
